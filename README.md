Плагин mpv-launcher для Raycast на Windows

Как установить:
1. Установить mpv
2. Добавить mpv в PATH
3. Установить и Node.JS и ffmpeg (для обложек)
```
winget install Node.js ffmpeg 
```
4. Клонировать репо
```
git clone https://github.com/zxc-rv/mpv-launcher && cd mpv-launcher
```
5. Установить плагин
```
npm install
```
```
npm run dev
```
